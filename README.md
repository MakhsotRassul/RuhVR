# RuhVR
Аппаратно-программный комплекс RuhVR для интерактивной оцифровки и сохранения культурного наследия
Категория соревнований: WRO 2026 Future Innovators (Старшая возрастная группа)
Тема сезона: Роботы и культура (Robots Meet Culture)
Направление разработки: Защита, сохранение и распространение культурного наследия
1. Введение, актуальность и проблематика проекта
Проект RuhVR представляет собой интегрированную робототехническую экосистему, разработанную для решения проблемы утраты интереса молодежи к изучению истории и физического разрушения памятников сакральной географии. В качестве объекта оцифровки и визуализации выбрана эпоха Казахского ханства (XV–XVIII века), включающая архитектурные объекты, такие как мавзолеи и древние городища, а также предметы материальной культуры. Синергия антропоморфной робототехники, виртуальной (VR) и дополненной (AR) реальности позволяет перевести образовательный процесс из пассивного заучивания в формат интерактивного исследования.
2. Архитектура и функциональные модули экосистемы
Комплекс состоит из трех технологических блоков, функционирующих в единой связке:
Физический антропоморфный робот-гид. Выполняет роль интерактивного интерфейса для взаимодействия с пользователями в реальном мире. Корпус робота стилизован под элементы национальной одежды (шапан, такия) для отражения культурной идентичности. Механическая часть базируется на узлах концептов pib и InMoov.
Иммерсивная VR-платформа. Разработанная трехмерная среда, обеспечивающая полное погружение пользователей в реконструированные исторические локации.
Система мобильных AR-карточек. Инструмент для вывода высокоточных цифровых двойников (Digital Twins) музейных артефактов и оружия на экраны мобильных устройств при распознавании маркеров.
3. Инженерная спецификация и кинематика робота
Механическая структура робота RuhVR спроектирована по гуманоидной схеме и включает исполнительные механизмы с несколькими степенями свободы:
Манипулятор правой руки (5 степеней свободы): Оснащен пятью независимыми сервоприводами. Каждый привод через систему механических тяг управляет отдельным пальцем, что позволяет генерировать жесты приветствия, указания на экспонаты и элементы демонстрации.
Локтевой сустав (1 степень свободы): Приводится в движение сервоприводом с повышенным крутящим моментом для обеспечения вертикального подъема и опускания предплечья.
Шейный узел (1 степень свободы): Включает сервопривод для автоматического поворота головы по горизонтальной оси с целью ориентации на пользователя.
Мобильное шасси: Дифференциальная ходовая платформа на базе двух независимых моторов постоянного тока с редукторами для перемещения робота по ровной поверхности.
4. Электроника, сенсорика и распределение питания
Вся электронная начинка полностью размещена внутри корпуса робота:
Вычислительное ядро нижнего уровня: Встроенная микроконтроллерная плата Arduino или ESP32, отвечающая за обработку сенсорных данных и расчет углов движения приводов.
5. Реализация стопроцентной автономии согласно Правилу 5.1 WRO
В строгом соответствии с регламентом WRO Future Innovators, робот RuhVR функционирует полностью автономно. Конструкция и программное обеспечение исключают использование внешних пультов управления, радиосвязи, смартфонов или команд со стороны человека.
Управление ходовой базой и манипуляторами осуществляется исключительно встроенной в микроконтроллер программой. В процессе движения робот автоматически анализирует расстояние до окружающих объектов с помощью ультразвуковых датчиков. При обнаружении человека на определенном расстоянии алгоритм самостоятельно останавливает моторы шасси, поворачивает шею, активирует сервоприводы пальцев для генерации приветственного жеста и посылает сигнал на запуск сценария вывода эмоций на встроенном экране.
6. Структура репозитория для судейской коллегии
Папка documentation: Содержит научно-исследовательское обоснование исторической точности, анализ социального эффекта проекта и проработанную бизнес-модель образовательного стартапа.

Папка hardware: Содержит инженерные спецификации исполнительных механизмов, электроники, ходовой базы и логику распределения питания.

Папка src: Содержит исходный функциональный код для прошивки робота и скрипты управления дополненной реальностью.
